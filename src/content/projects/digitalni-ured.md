## Digitalni ured

Digitalni ured je sveobuhvatna platforma za digitalizaciju poslovnih procesa namijenjena tvrtkama koje žele modernizirati način upravljanja dokumentacijom.

### Ključne značajke

- **Upravljanje dokumentacijom**: Jednostavno dodavanje, uređivanje i organizacija dokumenata
- **Elektroničko potpisivanje**: Sigurna integracija s sustavima elektroničkog potpisivanja
- **Automatizacija procesa**: Automatski tijekovi odobravanja i obavijesti
- **Integracije**: Jednostavna povezivanost s postojećim poslovnim sustavima
- **Sigurnost**: Robustni sigurnosni mehanizmi i backup sustavi

### Tehnički stack

- **Backend**: Python s FastAPI frameworkom
- **Baza podataka**: PostgreSQL za trajno spremanje podataka
- **Predmemoriranje**: Redis za brzi pristup često korištenim podacima
- **Kontenerizacija**: Docker za jednostavno deployanje i skaliranje
- **API**: RESTful API dizajn za lakšu integraciju

### Rezultati

Platforma je omogućila klijentima smanjenje papirnog rada za više od 80%, uz značajno ubrzanje poslovnih procesa i poboljšanu sigurnost dokumenata.
